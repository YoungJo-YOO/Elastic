## 💬프로젝트 설명
- 기후와 주요요인 중 실제로 농산물의 가격에 영향을 많이 주는 요소를 찾고, 그 요소들로 농산물의 가격을 예측한다

### 💬선정 근거 
농작물에 따라 생산량,수입량이 다 다르기에 국내요인 뿐만 아니라 외부요인도 고려가 필요하다 판단함
